# 知学汇 - 在线智慧教育平台

🚀 基于微服务架构的在线智慧教育平台，提供全方位的学习解决方案

## 📖 项目简介

本项目是一款大型在线学习网站，基于 **SpringCloud Alibaba** 微服务技术栈，集成在线教育、社交、电商等服务。引入 **Spring AI 框架**构建智能 **Agent** 服务，为学生端提供个性化课程推荐，为管理端提供自动化运营助手（周报/月报生成、数据查询）。学生端核心业务包括视频点播、积分排行、互动问答、课程购买等；管理端核心业务包括用户管理、课程管理、优惠券管理等。

### 核心价值

- **一站式教育解决方案**：支持多种学习场景和教学模式
- **微服务架构**：模块化设计，独立部署，弹性扩展
- **智能化学习**：基于AI的课程推荐和智能助手
- **数据驱动**：学习数据统计与分析，个性化学习路径
- **高可用设计**：服务容错，熔断降级，分布式事务

### 中间件

- RabbitMQ：消息队列

### 工具库

- MyBatis-Plus 3.4.3：ORM框架
- Hutool 5.7.17：Java工具库
- Swagger 3.0.3：API文档
- Lombok 1.18.20：代码简化工具

## 🚀 快速开始

### 环境要求

- JDK 11+
- Maven 3.6+
- MySQL 8.0+
- Redis 6.0+
- Nacos 2.0+
- RabbitMQ 3.8+

### 本地开发

1. **克隆项目**
   ```bash
   git clone https://github.com/EvanYao826/zhixuehui-platform.git
   cd zhixuehui-platform
   ```
2. **配置环境**
   - 复制各服务的`bootstrap-example.yml`文件为`bootstrap.yml`
   - 填写相应的环境变量或配置信息
   - 配置Nacos地址、数据库连接、Redis连接等
3. **编译项目**
   ```bash
   mvn clean install -DskipTests
   ```
4. **启动服务**
   - 先启动Nacos服务
   - 依次启动各微服务
   - 启动网关服务
